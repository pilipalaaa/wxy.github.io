《Web应用基础》课程结业报告


    本次我做了一个能源管理系统，利用网上的数据做出了一个首页、监测能耗、统计查询、系统管理等功能的页面。
其中能耗监测包括小时电量数据、小时实时数据、小时有功功率、小时功率因素、每日电量数据和每月电量数据。统
计查询包括小时电量统计和时段电量统计。还有一个系统管理页面，但查询功能还未实现。
    主要使用了layui框架。
    遇到的问题①：tabel 报错 The ID option was not found in the table instance。
    解决方法：配置表格时要添加表格的id。
    问题②：在做批量删除的时候,存储的数据为数组,后台要求接收的参数也是数组,但是请求的时候并没有成功
    解决方法：这是因为后台不识别数组,要把它转换成对象格式,比如:idStr.join(','),这样就把数组转换成对象,
成功传递给后台进行数据交互了。
    问题③：如何在layui里实现导出功能。
    解决方法：暂时没有成功解决。
    未解决的问题：查询与导出功能还未完善。
    总结：本次做的页面让我收获颇丰，不仅学习了layui框架的基本知识，还运用到了页面上。而且不懂的许多问题
也通过查阅网上资料解决了，比如layui使用过程中要注意代码执行的顺序，执行的顺序会导致有时候回显不上去的情
况。还有layui表单类，select要用form.render('select')   其他的一定要用form.render()  来刷新列表，否则
会出现显示不出来的情况等。所以此次页面虽然完成，但仍有许多不足，我也会继续学习相关知识来完善。
